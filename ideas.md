# 비가 그친 방에는 늘 네가 산다 - 디자인 철학

## 선택된 디자인 접근: 감성적 미니멀리즘

### 디자인 철학
이 페이지는 **애절한 감정을 시각화**하는 것에 중점을 둡니다. 사용자가 노래의 가사와 감정에 몰입할 수 있도록 설계되었습니다.

### 핵심 원칙
1. **감정 중심**: 이미지와 타이포그래피가 노래의 슬픔과 그리움을 강화
2. **공간의 힘**: 여백을 활용하여 각 가사 구절에 숨 쉴 공간 제공
3. **차분한 톤**: 과도한 장식 없이 순수한 감정 전달
4. **시각적 계층**: 제목, 가사, 음악 플레이어가 명확하게 구분됨

### 색상 철학
- **주 색상**: 깊은 파란색/회색 톤 (감정의 깊이와 차분함)
- **배경**: 밝은 회색 또는 흰색 (명확성과 순수함)
- **강조 색상**: 부드러운 파란색 (그리움과 거리감)
- **텍스트**: 어두운 회색/검은색 (높은 가독성)

### 레이아웃 패러다임
- 세로 스크롤 중심의 선형 구조
- 이미지가 상단을 차지하는 히어로 섹션
- 가사가 중앙에 배치되는 명상적 레이아웃
- 하단의 음악 플레이어 및 다운로드 버튼

### 시그니처 요소
1. **이미지**: 비 그친 방의 감성적 일러스트레이션
2. **타이포그래피**: 가사 구절 사이의 공간과 라인 높이
3. **음악 플레이어**: 미니멀한 디자인의 오디오 컨트롤

### 상호작용 철학
- 부드러운 호버 효과 (버튼 위에서 미묘한 색상 변화)
- 스크롤 시 이미지와 텍스트의 자연스러운 흐름
- 다운로드 버튼의 명확한 CTA

### 애니메이션 가이드
- 페이지 로드 시 이미지와 텍스트의 부드러운 페이드인
- 호버 시 버튼의 미묘한 스케일 변화 (100ms)
- 스크롤 시 자연스러운 패럴랙스 효과 (선택사항)

### 타이포그래피 시스템
- **제목**: 큰 세리프 또는 우아한 산세리프 (감정 전달)
- **가사**: 읽기 편한 산세리프, 넉넉한 라인 높이 (1.8-2.0)
- **메타 정보**: 작은 산세리프, 회색 톤

### 브랜드 본질
**한 줄 포지셔닝**: 비 그친 후에도 남겨진 그리움을 감정적으로 표현하는 노래

**성격**: 애절함, 진정성, 성찰적

### 브랜드 음성
- 헤드라인: "비가 그친 이 방에는 늘 네가 산다"
- CTA: "노래 듣기", "다운로드"
- 미니 카피: 가사 구절 인용

### 로고 및 아이콘
- 간단한 빗방울 또는 창문 아이콘
- 페이지 헤더에 배치

## 구현 계획

### 섹션 구조
1. **헤더**: 로고 및 제목
2. **히어로 이미지**: 비 그친 방의 감성적 일러스트레이션
3. **곡 정보**: 제목, 설명
4. **가사 섹션**: 전체 가사 (구절별로 구분)
5. **음악 플레이어**: 오디오 컨트롤 및 다운로드 버튼
6. **푸터**: 간단한 정보

### 기술 스택
- React 19 + Tailwind CSS 4
- shadcn/ui 컴포넌트
- 외부 오디오 플레이어 (HTML5 audio)

### 색상 팔레트
- 배경: `oklch(0.98 0.001 286.375)` (밝은 회색)
- 텍스트: `oklch(0.235 0.015 65)` (어두운 회색)
- 강조: `oklch(0.623 0.214 259.815)` (부드러운 파란색)
- 이미지 배경: 어두운 톤 (감정 강화)
